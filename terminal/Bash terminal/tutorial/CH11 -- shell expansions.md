# CH11 -- shell expansions
## objectives
You will know

    + brace expansion
    + tilde expansion
    + parameter and variable expansion
    + command substitution
    + arithmetic expansion
    + word splitting
    + filename expansion
    + quote removal
    + priority of these expansions

## CH11-1 -- priority of these expansions

The priority from highest to lowest are as follows

    + brace expansion (done in inside-to-outside fashion)
    + tilde expansion
    + parameter and variable expansion
    + command substitution (done in left-to-right fashion)
    + arithmetic expansion (done in inside-to-outside fashion)
    + word splitting
    + filename expansion
    + quote removal
    + priority of these expansions
