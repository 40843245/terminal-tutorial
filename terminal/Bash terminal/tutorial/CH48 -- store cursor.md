## objectives

## CH48-1 -- store mouse cursor
To store the mouse cursor,

type 

```
set-mark
```

or press the `Ctrl` + `@` (by default)

on terminal to store the mouse cursor.

For more details, see [`set-mark` on Gemini response](https://gemini.google.com/share/b4b457319eda)
