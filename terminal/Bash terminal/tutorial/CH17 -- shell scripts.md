## CH17-1 -- execute shell script
You can simply enter a file name of shell script (with arguments if necessary)

or enter built-in command `bash` followed by file name of shell script (with arguments if necessary)

syntax:

```
<file-name> {arguments}?
```

or

```
bash <file-name> {arguments}?
```
