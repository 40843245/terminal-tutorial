# CH7 -- comparison
## objectives
You will know how to 

  + compare two integers.

## CH7-1 -- comparison of two integers
> [!NOTE]
> You can't compare two numbers using symbol such as `!=`.
>
> You have to use flag (like in IL) to do so.

| flag | description |
| :-- | :-- |
| `-eq` | equal to |
| `-ne` | not equal to |
| `lt` | less than |
| `gt` | greater than |
| `le` | less than or equal to |
| `ge` | greater than or equal to |
